# BitHoard 功能列表

> 磁链资源管理与下载工具 · 版本 0.1.0

---

## 1. 认证与访问控制

### JWT 登录与会话管理
- 管理员单密码登录，签发 JWT 令牌
- API 路由鉴权中间件，保护所有非公开接口
- Web 端持久化登录态，支持手动退出

### IP 白名单 / 远程访问保护
- 基于客户端 IP 的访问控制
- 默认允许本地访问（127.0.0.1、::1、localhost）
- 可配置白名单，阻挡未授权的远程请求

---

## 2. 资源库管理

### 链接导入资源
- 支持 Magnet 链接、Torrent URL、ed2k 链接
- 剪贴板自动检测，批量捕捉并提示导入
- 支持手动输入链接创建资源

### 资源 CRUD 与元数据编辑
- 创建、查看、编辑、软删除、彻底删除资源
- 元数据字段：标题、描述、分类、状态、评分、评价
- 列表视图 / 卡片视图切换

### 资源状态工作流
- 状态生命周期：`draft` → `active` 等
- 界面中可视化展示当前状态

### 截图管理
- 为资源上传截图
- Sharp 自动生成缩略图
- 资源详情页缩略图画廊展示

### Torrent 文件与文件列表缓存
- 缓存 `.torrent` 文件内容
- 解析种子内文件列表及元数据
- 在详情页展示文件清单

### TMDB 自动匹配
- 根据资源标题自动查询 TMDB（电影/电视剧）
- 返回海报、概述、评分、类型、媒体类型等信息
- 用于资源元数据自动补全

### 操作历史 / 审计日志
- 记录资源的创建、更新、删除等操作
- 记录下载启动等事件
- 数据表 `history` 持久化存储

---

## 3. 搜索与发现

### 资源列表筛选与分页
- 按状态、分类、评分、标签、分组筛选
- 自由文本搜索
- 排序、分页

### 全文与文件名搜索
- 搜索资源标题、描述、磁链、评价字段
- 按缓存文件路径匹配资源

### 高级搜索
- 组合条件：状态、分类、评分范围、来源应用、标签、分组
- 是否有下载记录等筛选维度

---

## 4. 下载管理与 qBittorrent 集成

### 下载记录管理
- 创建、列表、更新、删除下载记录
- 下载记录与资源关联

### 从资源创建下载
- 验证磁盘空间
- 连接 qBittorrent 并添加种子
- 记录新下载任务

### qBittorrent 连接与状态
- 测试连接、显示版本信息
- 下载页面展示连接状态指示

### qBittorrent 种子控制
- 列出种子任务及文件列表
- 暂停 / 恢复 / 删除任务（可选删除文件）

### qBittorrent 元数据获取
- 获取种子属性和文件列表
- 缓存到本地数据库

### WebSocket 实时进度推送
- 定时轮询 qBittorrent 进度
- WebSocket 广播给所有连接客户端
- 下载页面实时进度条更新

---

## 5. 标签与分组

### 标签管理
- 创建、列表、删除标签
- 关联 / 取消关联资源

### 分组管理
- 创建、列表、更新、删除分组
- 添加 / 移除资源到分组

> ⚠️ 标签和分组的 Web UI 界面尚未完全实现，后端 API 已就绪。

---

## 6. 桌面应用 / 剪贴板 / 托盘

### 剪贴板监控
- 实时监测系统剪贴板中的磁链/Torrent/ed2k 链接
- 短时批量去重，发送到 UI 作为可操作的通知卡片

### 系统托盘与窗口管理
- 最小化到系统托盘而非退出
- 托盘图标点击恢复窗口
- 支持应用完全退出

### 全局快捷键
- 注册全局热键，一键切换 BitHoard 窗口显示/隐藏

### Electron 预加载桥接
- 安全的 contextBridge 暴露剪贴板事件和应用版本信息
- 渲染进程通过预加载脚本与主进程通信

---

## 7. Web 界面

### 应用外壳与导航
- 侧边栏主导航：全部资源、下载管理、设置
- 路由式页面结构（svelte-spa-router）
- 快速统计面板（总资源数）

### 首页（全部资源）
- 资源搜索栏
- 列表 / 卡片视图切换
- 资源计数、加载状态、空状态

### 资源详情页
- 完整元数据展示
- 磁链 URI 及复制
- 截图缩略图画廊
- 文件列表
- 标签展示
- 下载状态及操作
- 编辑元数据入口

### 下载管理页
- 下载任务列表
- 状态 / 进度实时展示
- qBittorrent 连接状态指示
- 删除下载记录

### 设置页
- qBittorrent 连接配置与测试
- IP 白名单信息展示
- 退出登录

### Toast 通知与暂存区
- 剪贴板检测到的链接以 Toast 卡片呈现
- 可编辑、可选择导入
- 暂存区管理待导入资源

## 8. 项目脚本与配置

### 开发脚本

| 命令 | 说明 |
|------|------|
| `pnpm dev` | 同时启动 Server + Web + Electron |
| `pnpm dev:server` | 单独启动 Express 服务端 |
| `pnpm dev:web` | 单独启动 Vite 前端开发服务器 |
| `pnpm dev:electron` | 单独启动 Electron 桌面应用 |
| `pnpm build` | 构建 Web UI + Electron 打包 |

### 环境变量配置

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `PORT` | 服务器端口 | `13002` |
| `HOST` | 绑定地址 | `127.0.0.1` |
| `JWT_SECRET` | JWT 签名密钥 | — |
| `ADMIN_PASSWORD` | 管理员登录密码 | `admin` |
| `QBITTORRENT_URL` | qBittorrent 地址 | `http://localhost:8080` |
| `QBITTORRENT_USERNAME` | qBittorrent 用户名 | `admin` |
| `QBITTORRENT_PASSWORD` | qBittorrent 密码 | — |
| `TMDB_API_KEY` | TMDB API 密钥 | — |
| `IP_WHITELIST` | IP 白名单（逗号分隔） | `127.0.0.1,::1,localhost` |
| `DATA_DIR` | 数据存储目录 | `data/` |
